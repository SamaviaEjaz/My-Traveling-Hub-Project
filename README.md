# Traveling-Hub 

A React Native project for travel booking and ride-sharing.

## Installation
1. https://github.com/SamaviaEjaz/My-Traveling-Hub-Project.git
2. Run `npm install`
3. Start with `npx expo start`

## Features

##
Admin Login 
- DriverApproval
- Delete Driver 
- ViewComplaints

##
- Passenger login
- Chat system
- Complaint system
- GetRide and Book Ride 
- ViewBookingStatus 

## 
- Driver Register & login
- SharePost 
- ViewBookingStatus
- ReceiveRequest 
- Get Reviews
